# Coding Challenge: Aplicación CRUD React

## Requerimientos

Crear una aplicación React (CRA o Vite) que implemente una funcionalidad CRUD para una lista de productos.

### Modelo Producto
```json
{
  "id": "string",
  "nombre": "string",
  "precio": "number",
  "cantidad": "number"
}


Características:
Listado de productos en una tabla.


Formulario modal para crear/editar productos.


Botón “Eliminar” con confirmación.


Manejo de estado con hooks (useState, useReducer).


Validación de formularios (react-hook-form o validaciones manuales).


Consumo de una API de productos (simulada con JSON Server o Mock API).


Notificaciones de éxito/error al crear, editar o eliminar.


Diseño estilizado con Tailwind CSS.


Pruebas unitarias mínimas:
Al menos 2 tests de componentes (p. ej., validación de formulario).


Al menos 2 tests de funciones auxiliares (p. ej., cálculo de totales).


Entregable:
Repositorio Git con carpeta de proyecto y README con instrucciones de instalación, ejecución y ejemplos de uso.
